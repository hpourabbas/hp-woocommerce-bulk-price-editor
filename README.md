---

## Description

Managing prices in a WooCommerce store can become time-consuming and error-prone when you need to update dozens or hundreds of products.  
**HP WooCommerce Bulk Price Editor** is built to solve this exact problem.

This plugin allows store owners to **update product prices and apply discounts in bulk**, eliminating the need to edit products one by one. Whether you are running a seasonal sale, adjusting prices due to market changes, or applying store-wide discounts, this plugin helps you do it **quickly, safely, and consistently**.

With a simple and focused interface, you can:
- Modify prices of multiple WooCommerce products at once
- Apply bulk discounts without affecting unrelated products
- Save time and reduce human error in price management
- Keep your store pricing strategy flexible and easy to maintain

**HP WooCommerce Bulk Price Editor** is lightweight, straightforward, and designed for store owners who want **full control over pricing without unnecessary complexity**.

---

## Installation

### Installation from within WordPress

1. Go to **Plugins > Add Plugin**.
2. Search for **HP WooCommerce Bulk Price Editor**.
3. Install and activate the plugin.

### Manual installation

1. Upload the `hp-woocommerce-bulk-price-editor` folder to `/wp-content/plugins/`.
2. Go to **Plugins** in the WordPress admin panel.
3. Activate the plugin.

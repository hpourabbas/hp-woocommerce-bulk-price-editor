Changing the price of WooCommerce products and applying discounts to products in bulk

== Description ==
With **HP Woocommerce Bulk Price Editor** you can change the price of WooCommerce products and apply discounts to products in bulk

== Installation ==

= Installation from within WordPress =

1. Visit **Plugins > Add Plugin**.
2. Search for **HP Woocommerce Bulk Price Editor**.
3. Install and activate the **HP Woocommerce Bulk Price Editor** plugin.

= Manual installation =

1. Upload the entire `hp-woocommerce-bulk-price-editor` folder to the `/wp-content/plugins/` directory.
2. Visit **Plugins**.
3. Activate the **HP Woocommerce Bulk Price Editor** plugin.
